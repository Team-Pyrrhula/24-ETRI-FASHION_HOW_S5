## 대회 개요
> ...(중략)... 이번 대회는 **모델 경량화**에 초점을 맞추어 실시간 성능과 효율성을 높이는 데 중점을 둡니다. ...(중략)... FASHION-HOW 시즌5는 4개의 세부 과제가 운영됩니다. 참여하시는 분들은 4개 세부 과제에 모두 참여가 가능하며, **4개 세부 과제의 통합 성적으로 우승자를 선정**합니다.
